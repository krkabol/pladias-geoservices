# pladias-geoservices
JS module for consuming [Pladias](pladias.cz) geowebservices by using OpenLayers. 


### sample usage in package.json
```json
{
  "dependencies": {
    "pladias-geoservices": "krkabol/pladias-geoservices#v0.0.1"
  }
}
```
